Como Usar

1. Para mudar o tema: Adicione a classe correspondente na tag <html>

2. Exemplo: <html class="theme-light"> para tema claro

3. Classes disponíveis:

  • theme-light - Tema claro
  • theme-yellow-dark - Amarelo/Preto
  • theme-brown-light - Marrom/Branco
  • theme-purple-orange - Roxo/Laranja

4. Classes semânticas: Use text-foreground, bg-card, border-border, etc.
